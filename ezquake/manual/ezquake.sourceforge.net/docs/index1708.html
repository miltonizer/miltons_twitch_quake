ezQuake Manual: Demo Player

Select 'Multiplayer' in the Main menu and then 'Demos' to access the Demo Player menu. Use your arrow keys [Left] and [Right] to move accross three main Demo Player menu sections. Use [Tab] to select different sub-menus.

Demos menu

You can browse the directory structure on your drive using the [Arrow Up], [Arrow Down], [Page Up] and [Page Down]. You can play the demo by hitting the [Enter] key. You can add the demo to the playlist using [Ctrl]+[Enter].
You can delete demos from you drive from this menu too. Use [Del] to remove demos from your drive with confirmative question ([Y] to confirm the deletion) or [Shift]+[Del] to remove the demo from your drive instantly.

Sort the demos by name by pressing [Ctrl]+[N], by time by pressing [Ctrl]+[T] and by size by pressing [Ctrl]+[S].

Playlist menu

Demos submenu

You can browse the demos in the playlist, change the order using [Ctrl]+[Arrow Up] and [Ctrl]+[Arrow Down] or delete the demo from the playlist (doesn't delete the demo physically) using [Del]. You can start the demo playback by selecting the first demo in the playlist and hitting [Enter]. Of course you can start the demoplay selecting any demo you want.

Options submenu

Using first four options you can jump through played demos, stop the playback of the demos and clear the playlist. Then you can see which demo is currently selected in the Demos submenu of Playlist menu. You can set which player the client should start tracking on the start of the playback of the selected demo. Type part of the desired nickname into the 'Track' textfield.

Options menu

Here you can turn looping of the playlist ON or OFF and set the default track value. Default track is a part of nickname of the player who the client will automatically start tracking after the demo playback starts. This setting has lower priority than the 'Track' field in the Playlist menu.

Navigation summary

  • Use your arrow keys [Left] and [Right] to move accross three main Demo Player menu sections.
  • Use [Tab] to select different sub-menus.
  • Use arrow keys [Up] and [Down] to move across demos or items from each menu.
  • Use [Enter] to select demos in the Demos or Playlist menu.
  • Use [Del] to remove demos from your drive in the Demos menu or demos from the Playlist menu.
  • Use [Ctrl]+[Arrow Up] and [Ctrl]+[Arrow Down] to rearrange demos in the playlist menu.
  • Use [Ctrl]+[Enter] to add demo from the Demos menu to the Playlist.
