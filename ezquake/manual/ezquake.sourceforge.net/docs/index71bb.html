ezQuake Manual: Input - Joystick Variables

joystick 

Description

Toggles usage of joystick device.

Support: Windows: OpenGL Windows: Software Linux: GLX Linux: X11 Linux: SVGA Mac OS X FreeBSD

Default: 0

Values

valuedescription
1Use joystick input.
0Do not use joystick.
