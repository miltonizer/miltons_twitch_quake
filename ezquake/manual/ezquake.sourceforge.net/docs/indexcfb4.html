ezQuake Manual: Changelog 1.8.2

ezQuake 1.8.2 stable - 27th May 2007

Major changes
  • more server browser colors - feature by johnnycz
  • cl_newlerp - new interpolation of rockets, grenades and other entities, enabled by disconnect - feature by qqshka
  • cl_predict_half - eliminates player models' jittering when independent physics is enabled - feature by tonik
  • gl_textureless for brush models - fixed by tei
  • scr_sshot_format fix for movies capturing - fixed by cokeman
  • Intel cards flickering bugfix - fixed by cokeman
Other changes
  • show and hide commands list hud elements now sorted alphabetically - feature by disconnect
  • ctrl+w gives white led character - feature by up2nogood
  • gl_part_telesplash 2 - feature by qqshka
  • support both -norjscripts and -noscripts cmdline options - feature by disconnect
  • access console from menu with tilde key - feature by dkure
  • protection of when fakeshaft, scripts and textureless can be set - changed by disconnect
  • tp_name_teammate default to empty string so that player nick can be pointed - changed by disconnect
  • hud_rankingpos behavior fixed - fixed by cokeman
  • Z-fighting fix for gl_ztrick default to 0 - fixed by cokeman
  • fixed auth_viewcrc - fixed by disconnect
  • con_particles_images 0 fix - fixed by qqshka
  • Mac: 32 bit support fix, missing paks error fix - fixed by tuna
  • in-built teamplay messages runes support - fixed by up2nogood
  • demo player/menu fixes - fixed by cokeman, johnnycz
  • ruleset smackdown option linux crash bugfix - fixed by disconnect
  • x11 video options menu crash bugfix - fixed by disconnect
  • messagemode-menu loop lock - fixed by johnnycz
  • disable ICMP pings on Vista - changed by disconnect
