Demo Browser

Typing...Look-up demo
EnterPlay demo
Ctrl+2Show/hide demo size column
Ctrl+3Show/hide demo date column
Ctrl+4Show/hide demo time column
Alt+1Sort demos by name
Alt+2Sort demos by size
Alt+3Sort demo by date-time
Alt+4?
DelDelete demo (with confirmation)
Shift+DelDelete demo (immediately)
Shift+EnterBenchmark using given demo (timedemo)
Ctrl+EnterAdd demo to the playlist

Server Browser

SpaceRefresh the list (ping & update again)
Typing...Search for server with given name
EnterOpen up server details tab
Ctrl+CCopy server's IP address into the clipboard (Windows only)
Ctrl+VSend the server's IP address as a message to your teammates
Ctrl+[num]Show/hide column
Alt+[num]Sort by column (Alt+3 = Sort by ping)
Columns
1name
2address
3ping
4gamedir
5map
6players
7fraglimit
8timelimit

Console

Left Arrow
Right Arrow		Move cursor through the current line
End / HomeMove cursor to the beginning/end of the line
Up Arrow
Down Arrow		Browse commands history
Page Up
Page Down		Scroll console history up / down
Ctrl+Page Up
Ctrl+Page Down		Scroll console history by pages up / down
Ctrl+Up Arrow
Ctrl+Down Arrow		Decrease / Increase console size
Ctrl+Home
Ctrl+End		Jump to the beginning / end of the console history

HUD Editor

PToggle HUD Planmode
HToggle Help
F1Toggle Moving Elements
F2Toggle Resizing
F3Toggle Aligning
F4Toggle Placing
SpaceToggle Outlining
AltAlign current dragged element
CtrlPlace current dragged element
ShiftLock moving of current dragged element to X/Y axis
