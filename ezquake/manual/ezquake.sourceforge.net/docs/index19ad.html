ezQuake Manual: Install instructions

Requirements

To play QuakeWorld you need original Quake game with 1.06 patch.
Note: You don't need all the files from Quake. See Clean Quake Install below on this page.

Installation

After you download ezQuake extract/install the client into your Quake directory.

Upgrading from previous version of this client

See Upgrading manual page.

Already have fQuake pack

fQuake has been replaced by nQuake. But if you already have QuakeWorld installed using fQuake pack and you would like to use this client, just install it into your fQuake directory. The fQuake pack comes with ezQuake included by default though.

Already using FuhQuake

You can keep majority of your configuration and settings from FuhQuake by just copying / renaming the fuhquake directory in your Quake directory to ezquake.

After this, install / extract ezQuake into your Quake directory.

Troubleshooting

Most of the problems are caused by wrong files placement. Here we will make sure that you have all the files you need on right places.

Clean Quake Install

As we've said already, you need data from original Quake game to run this client. Those data are stored in files pak0.pak (included in shareware version of Quake) and pak1.pak (included in the full version of Quake game). To make a clean Quake installation do following:

  1. Create a new directory on your disc, we will call this directory your "Quake directory" or "Quake root" from now on.
  2. Create directory called "id1" in your Quake directory.
  3. copy both pak0.pak and pak1.pak files into the id1 directory

Now you can install this client or any other Quake engine into the Quake directory. Have fun. :-)

A typical error message is about glide2x.dll missing. To get rid of this error, remove the file opengl32.dll from your Quake directory.

Required final structure

  • Quake [dir]
    • id1 [dir]
      • pak0.pak [file from full or shareware Quake]
      • pak1.pak [file from full Quake version, optional]
    • ezquake [dir]
      • pak0.pak [included in ezQuake package]
      • help.pak
      • ...
    • ezquake.exe
    • ezquake-gl.exe
    • ezstart.exe
