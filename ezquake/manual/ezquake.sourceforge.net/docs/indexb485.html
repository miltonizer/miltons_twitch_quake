ezQuake Manual: System Requirements

Minimum Specifications

  • Windows 98 or XP or Vista / Linux / Mac OS X / FreeBSD 6
  • Processor 400 MHz
  • 128 MB RAM
  • Any video card (3D acceleration not necessary)
  • 56 kbps connection

Recommended Specifications

  • Windows XP or Vista / Linux / Mac OS X / FreeBSD 6
  • Processor 2.0 GHz
  • 512 MB RAM
  • OpenGL compatible Graphics card with 256 MB memory (nVidia GeForce 5800 or better)
  • Broadband Internet Connection
  • Good mouse :-)
