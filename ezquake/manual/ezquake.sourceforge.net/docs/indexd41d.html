Features
  • Changelog / Feature list - what's new and cool in this client
  • Independent physics - render the game with as much frames per second (FPS) as your hardware allows.
  • Server Browser - improved MQWCL's Server Browser, most missing feature in FuhQuake.
  • Head Up Display - how to customize your in-game HUD, move, resize, group, show and hide screen elements.
  • Graphics - graphical settings based on FuhQuake's manual + full description of new AMFQuake + TEI commands/variables.
  • MultiView Demos (MVD) support: Screen Multiview - up to four point of views, Radar and Team hold bar, MVD Info, MVD Autotrack, MVD Status.
  • Display settings - change screen resolution, display frequency and others on-the-fly.
  • TCL Support - Tool Command Language support
  • Demo Player - browse your dirs and zip and gz archives with demos, delete demos; Demo playlist - create playlists, set the default tracked player, looping
  • MD3 Models - download some from QuakeWorld.ru server.
  • Video stream capture - instantly capture xvid/divx videos with mp3 sound and combine with Camera scripting
  • Weapon handling - you don't need complicated "boomstick scripts" anymore
  • Scripting - basic & advanced scripting possibilities.
  • Custom Frag And Tracker Messages - 'Foobar is on killing spree - 10 frags in a row!' and frag messages in separated window
  • Colored Text
  • Player skins
  • Fakeshaft
  • Linux: Real Time Clock, EVDEV support
  • Keymap - this feature allows you to use non-US keyboard layout
  • Link files, Custom Paks.
  • Luma textures, Qizmo spectator ignoring, Taskbar flash on activity, Fog, Message filter, Skybox groups, Quick Quit, Custom player tracking label, Wall, Floor and Ceiling and liquids custom colors, Spawn corona control, Integrated Help and Describe system, Gameclock with seconds, Intelligent opponents ignore, Commands history, Grenade flash control for demos, Teleport splash control, Centerprint messages positioning, Macro $lastip contains last ip address seen in console, Scripted calculations, own User dir, nickname highlighting, ...
  • Custom textures, console font and background, crosshair and skyboxes.
  • Teamplay macros from Qizmo; other macros
  • Security features - client authentication, movement scripting control, rulesets; models checking
  • Features inherited from FuhQuake
    • Configuration Manager - How to save and restore your settings.
    • Spectator Tracking - Description of commands that make spectator tracking more convenient.
    • Logitech Mice - How to enable extra buttons on your logitech mice.
    • Particle Effects - How to enable and customise different particle effects.
    • MP3 Player - How to control the winamp or xmms mp3 players from within the client.
    • Crosshairs - Everything you need to know about customising crosshairs.
    • Match Tools - How to enable automatic demo recording, console logging and screenshot taking.
    • TP Pointing - How to communicate what you are pointing at to your teammates.
    • Console fonts - guide for proper charset sizes
    • Linux Information - Information specific to Linux binaries.
    • QC Extensions - Information about QC extensions available in the inbuilt server.
