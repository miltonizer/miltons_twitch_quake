ezQuake Manual: Triggers

Trigger is common name for a special kind of action that gets executed when something happens in the game.

Typical usage of Standard, Teamplay and Team Fortress triggers is to create an alias with the name of the trigger running some action you need.
Example: alias f_mapend "screenshot" - this will make a screenshot on the end of each match.

Other kind of triggers are Security checks and Qizmo triggers. If you type the name of the trigger in the game chat, some clients will reply with a specific text.
Example: /say f_skins - clients should reply if they are using fullbright skins.
