ezQuake Manual: Custom paks

It is possible to use PAK-files with arbitrary names. When the directory is added to the search path first files pak0.pak, pak1.pak are loaded, then pak.lst is read and all the PAK-files listed there are loaded (one per line with extension). GL-version skips all files strting with "soft", soft-version skips files starting with "gl". If userdir is set to name, then finally name.pak is loaded.
